Globe Rendering: Memory Comparison

Open each demo in its own tab, then take a heap snapshot in DevTools to compare memory footprint independently.

Canvas 2D

Canvas 2D (Antialiased)

Pixel-by-pixel texture projection with bilinear sampling. Renders at 2x and downscales for antialiasing. CPU-bound, zero dependencies.

 Three.js + WebGL

Three.js (Day/Night/Atmos)

GPU-accelerated sphere with day/night texture blending, Fresnel atmospheric glow, and MSAA antialiasing. Uses custom GLSL shaders.
How to compare:
1. Open each link in a separate tab
2. Wait for globe to start rotating
3. DevTools → Memory → Take Heap Snapshot
4. Compare retained sizes
5. Check Activity Monitor for CPU usage per tab
