Reference: How these callouts work

Question: How should callouts behave?

A

Always visible (all markers show callouts)

Every marker on the visible side gets a callout label with a leader line pointing outward from globe center. Labels auto-arrange to avoid overlap. Best for dashboards with fewer markers (5-30).

B

On hover / click only

Callouts appear when you interact with a marker. Clean globe by default, callout pops out on interaction. Best for dense datasets (30+ markers).

C

Configurable per marker (default: always visible)

Each marker has a callout property: "always", "hover", "click", or "none". The scene creator decides. Default is "always" for the reference look.

My recommendation: Option C — gives content creators full control. Default "always" matches the reference look you showed.
