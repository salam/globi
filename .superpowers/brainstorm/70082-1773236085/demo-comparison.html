Memory & Performance Comparison

Open each demo in a separate tab and use Chrome DevTools → Memory → Take Heap Snapshot to compare

How to test

  1. Open each link below in a separate tab
  2. Open DevTools (Cmd+Option+I) → Memory tab
  3. Click "Take heap snapshot" after the globe is rendering
  4. Also check the Performance tab to see frame times
  5. Compare the stats shown below each globe
