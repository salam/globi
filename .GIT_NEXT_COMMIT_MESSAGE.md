feat: add earthBuilder with day/night shader and atmosphere

- Uniforms: sunDirection, atmosphereColor (default [0.3, 0.6, 1.0]).
- All GLSL written as inline template literals; no external .vert/.frag files.
- Added 6 unit tests in tests/earth-builder.test.js; all pass (97/97 suite-wide).
