# feat: add MarkerManager for Three.js marker sprites

Summary:

- Added MarkerManager class that creates/updates/disposes Three.js Sprite and Mesh objects for geographic markers.
- Supports dot, image, model, and text visual types.
- Uses OffscreenCanvas in browser; gracefully returns null texture in Node.js test environment.
- Caches image textures via TextureLoader; clears and rebuilds on each update() call.
- dispose() releases all GPU resources and clears internal marker map.
- Added 4 unit tests in tests/marker-manager.test.js; all pass.
