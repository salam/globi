# feat: add ArcPathManager with Line2 fat lines

Summary:

- Added ArcPathManager class for rendering great-circle arcs and multi-point paths on the globe using Three.js fat lines (Line2 + LineGeometry + LineMaterial).
- Arcs are tessellated via greatCircleArc (64 segments); paths are densified via densifyPath (8° max step).
- Supports configurable color, strokeWidth, and optional dash patterns per arc/path.
- Added 4 unit tests covering arc creation, path creation, clear-on-update, and clean disposal.
